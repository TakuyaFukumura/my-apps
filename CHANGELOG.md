# Changelog

このプロジェクトのすべての変更はこのファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に基づいており、
このプロジェクトは [Semantic Versioning](https://semver.org/lang/ja/) に従っています。

## [0.5.1] - 2026-02-27

### 変更

- `README.md` を現況に合わせて更新
    - アプリ概要をSQLite/Hello worldの説明から自作アプリ紹介サイトの説明に変更
    - 技術スタックからSQLite（better-sqlite3）を削除
    - 機能セクションのアプリ一覧画面のURLを `/apps` から `/` に変更
    - プロジェクト構造から `lib/database.ts`・`api/message/`・`data/` を削除し、現状に合わせて更新
    - APIエンドポイントセクションを削除
    - データベースセクションを削除
    - カスタマイズセクションからメッセージ変更の説明を削除
    - テストファイル構成を現状に合わせて更新（database.test.ts削除、page.test.tsx追加）
    - テストの特徴からデータベーステストの説明を削除
    - トラブルシューティングからデータベース関連エラーの説明を削除

## [0.5.0] - 2026-02-27

### 変更

- トップページ（`/`）のリダイレクトを廃止し、アプリ一覧を直接表示するよう変更
- `/apps` ルート（アプリ一覧ページ）を廃止
- アプリ詳細ページの「アプリ一覧に戻る」リンクを `/apps` から `/` に変更
- ヘッダーの「アプリ一覧」ナビゲーションリンクを `/apps` から `/` に変更

### 削除

- `docs/remove-redirect-design.md`（実装完了のため削除）

## [0.4.0] - 2026-02-27

### 追加

- 全アプリにデプロイ先URL（`siteUrl`）を設定
  - 配当可視化アプリ: https://divichart-next-js-app.vercel.app/
  - 資産可視化アプリ: https://visuasset-next-js-app.vercel.app/
  - Coast FIRE 必要資金計算アプリ: https://coast-fire-next-js-app.vercel.app/
  - ブログアプリ: https://blog-next-js-app-taupe.vercel.app/

### 変更

- アプリ一覧画面からGitHubリンクを削除し、デプロイ先リンクのみ表示するよう変更

## [0.3.0] - 2026-02-27

### 変更

- ホーム画面を削除し、ルートパス (`/`) をアプリ一覧画面 (`/apps`) へリダイレクトするよう変更
- DB関連処理（SQLite・`/api/message` エンドポイント・`lib/database.ts`）を削除
- `better-sqlite3` 依存パッケージを削除
- レイアウトのメタデータ（タイトル・説明文）をアプリ一覧に合わせて更新

## [0.2.0] - 2026-02-26

### 追加

- 自作アプリ紹介画面（一覧・詳細）を新規実装
    - アプリ一覧ページ (`/apps`)：作成者情報セクション・レスポンシブタイルグリッド（1/2/3カラム）
    - アプリ詳細ページ (`/apps/[id]`)：詳細説明・GitHubリンク・オプションのサイトリンク
    - ビルド最適化のため `generateStaticParams` を詳細ページに追加
- ヘッダーに「アプリ一覧」ナビゲーションリンクを追加
- ヘッダーのロゴをトップページへのリンクに変更
- 絵文字アイコンへの `aria-hidden="true"` 追加（アクセシビリティ向上）
- `siteUrl` をオプション型にし、未設定時はサイトリンクボタンを非表示に変更
- 仕様書 `docs/app-showcase-spec.md` を追加
- アプリ一覧・詳細ページおよびヘッダーのテストを追加
- `README.md` の「機能」「プロジェクト構造」「テスト」セクションを更新

## [0.1.0] - 初期リリース

### 追加

- SQLiteからメッセージを取得するシンプルなNext.jsアプリケーション
- ダークモード対応ヘッダー
- API エンドポイント (`/api/message`)
