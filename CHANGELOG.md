# Changelog

このプロジェクトのすべての変更はこのファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に基づいており、
このプロジェクトは [Semantic Versioning](https://semver.org/lang/ja/) に従っています。

## [0.3.0] - 2026-02-27

### 変更

- ホーム画面を削除し、ルートパス (`/`) をアプリ一覧画面 (`/apps`) へリダイレクトするよう変更
- DB関連処理（SQLite・`/api/message` エンドポイント・`lib/database.ts`）を削除
- `better-sqlite3` 依存パッケージを削除
- レイアウトのメタデータ（タイトル・説明文）をアプリ一覧に合わせて更新

## [0.2.0] - 2026-02-26

### 追加

- 自作アプリ紹介画面（一覧・詳細）を新規実装
    - アプリ一覧ページ (`/apps`)：作成者情報セクション・レスポンシブタイルグリッド（1/2/3カラム）
    - アプリ詳細ページ (`/apps/[id]`)：詳細説明・GitHubリンク・オプションのサイトリンク
    - ビルド最適化のため `generateStaticParams` を詳細ページに追加
- ヘッダーに「アプリ一覧」ナビゲーションリンクを追加
- ヘッダーのロゴをトップページへのリンクに変更
- 絵文字アイコンへの `aria-hidden="true"` 追加（アクセシビリティ向上）
- `siteUrl` をオプション型にし、未設定時はサイトリンクボタンを非表示に変更
- 仕様書 `docs/app-showcase-spec.md` を追加
- アプリ一覧・詳細ページおよびヘッダーのテストを追加
- `README.md` の「機能」「プロジェクト構造」「テスト」セクションを更新

## [0.1.0] - 初期リリース

### 追加

- SQLiteからメッセージを取得するシンプルなNext.jsアプリケーション
- ダークモード対応ヘッダー
- API エンドポイント (`/api/message`)
