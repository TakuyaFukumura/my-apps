# 自作アプリ紹介画面 仕様書

## 概要

自作アプリを紹介するための画面を作成する。
アプリ一覧画面とアプリ詳細画面の2種類のページで構成される。

## 画面構成

### 1. アプリ一覧画面 (`/apps`)

#### レイアウト

- 画面上部に作成者情報を表示する
- アプリをタイル状（グリッドレイアウト）で並べて表示する
- スマートフォン対応：1カラム → タブレット：2カラム → PC：3カラム

#### 作成者情報セクション

| 項目 | 内容 |
|------|------|
| 作成者名 | TakuyaFukumura |
| GitHubプロフィールリンク | https://github.com/TakuyaFukumura |

#### アプリタイル

各タイルに以下の情報を表示する。

| 項目 | 説明 |
|------|------|
| アプリ名 | アプリの名称 |
| 簡易説明 | アプリの概要（短い説明文） |
| サイトリンク | アプリのサイトへのリンク |
| 詳細リンク | アプリ詳細画面へのリンク |

### 2. アプリ詳細画面 (`/apps/[id]`)

#### 表示内容

| 項目 | 説明 |
|------|------|
| アプリ名 | アプリの名称 |
| 詳細説明 | アプリの詳しい説明文 |
| サイトリンク | アプリのサイトへのリンク |
| GitHubリンク | GitHubリポジトリへのリンク |
| 一覧に戻るリンク | アプリ一覧画面へ戻るリンク |

## 技術仕様

### 使用技術

- Next.js（App Router）
- TypeScript
- Tailwind CSS
- 既存のダークモード機能を活用

### ファイル構成

```
src/app/apps/
├── data.ts          # アプリデータ定義
├── page.tsx         # アプリ一覧画面
└── [id]/
    └── page.tsx     # アプリ詳細画面
```

### レスポンシブ対応

| ブレークポイント | カラム数 |
|----------------|---------|
| モバイル（デフォルト） | 1カラム |
| タブレット（sm: 640px以上） | 2カラム |
| PC（lg: 1024px以上） | 3カラム |

## UIデザイン方針

- 既存アプリのデザインテイスト（青系カラー、丸みのあるカード）に統一する
- ダークモード対応
- ホバー時にカードを浮き上がらせるアニメーション効果を付与する
- タイルは均等なサイズで整列させる
